This WordPress Plugin Adds various widgets Elementor page builder to work with WHMCS which is used by most of the hosting companies to bill and manage clients and servers. If you want you can use our plugin with only your WHMCS or also with whmcs bridge plugin(integrates whmcs to wordpress and also real whmcs url from customers). 

You must have [Elementor Page Builder](https://wordpress.org/plugins/elementor/) installed.
You must have and url of whmcs from where and to where the data will be fetched from and given.
== Description ==

This plugin adds new element / widget such as: Domain Searcher, Live domain Searcher, Pricing table, Live pricing table which fetches pricing from your WHMCS directly so you wont have to change prices each time you update your pricing in WHMCS!, Non live mode for pricing table, Knowledge base box for searcing and posting the search to whmcs or whemcs bridge page - to Elementor Page Builder! It's and easy and handy plugin for you if you want to start hosting business and use wordpress site with elementor !

* Live Ajax Domain Search 
* Knowledge Base form for WHMCS
* Live pricing table from WHMCS
* Normal pricing table.

== Installation ==

1. Upload the plugin folder after extracting it to the "/wp-content/plugins/(the folder of the extracted plugin)" directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to elementor page builder mode and then you should see void elements if you scroll a bit in the elements of elementor. The live pricing, name etc will be shown only when you save the page and view it in the front end of the site.

== Frequently Asked Questions ==

= Where can to find the new element added =

Our Widgets For Elementor Page Builder adds a new section in the elementor live page builder named : VOID ELEMENTS. There you will see the widget which you can drag and drop and config settings

== Dependencies ==
The Plugin's domain search depends on api call to external free service provided by [http://whoiz.herokuapp.com/](http://whoiz.herokuapp.com/).
